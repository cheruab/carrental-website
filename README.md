# 🚗 Car Rental Website

A modern **Car Rental Website** that allows users to browse available vehicles, view detailed car information, and book cars for their travel needs.

The application is built using **React** for the frontend and **NestJS with GraphQL** for the backend, with **PostgreSQL** as the database.

---

## ✨ Features

* 🚘 **Car Listings** – Browse a variety of cars available for rent
* 📄 **Car Details** – View specifications, pricing, and information about each car
* 📅 **Booking System** – Users can rent cars for a selected time period
* ⚡ **GraphQL API** – Efficient data querying and mutations
* 📱 **Responsive Design** – Works seamlessly across desktop and mobile devices

---

## 🛠 Tech Stack

### Frontend

* React
* Tailwind CSS

### Backend

* NestJS
* GraphQL

### Database

* PostgreSQL
